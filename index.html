Install Remi on macOS

Development Build - Security Instructions Included
Download started! Follow the steps below to install Remi.
1

Download Started!

Your download should start automatically. If it doesn't, use the link below to download manually.

Download from GitHub
2

Extract & Move to Applications

Extract the downloaded zip file and drag Remi.app to your Applications folder.

Remi.app
Applications
3

Bypass Security Warning

Since this is a development build, macOS will show a security warning. Here's how to open it safely:

Right-click on Remi.app in Applications
Select "Open" from the context menu
Click "Open" in the security warning dialog
4

Launch & Enjoy

Remi will appear in your menu bar and remember your security choice for future launches.

Look for Remi in your menu bar
Global hotkey:
⌘⇧R
macOS will remember your choice - you only need to do this once!
